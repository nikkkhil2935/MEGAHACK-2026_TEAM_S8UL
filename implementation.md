## 1. Vision & Problem Statement

### The Problem
- LinkedIn / Naukri / Indeed use **keyword-only** matching — wrong jobs, wrong candidates
- No platform shows candidates **exactly which skills they're missing** for a specific role
- Mock interview tools are static — pre-recorded, no AI conversation, no cross-examination
- Zero anti-cheating in online interviews — tab switching, eye drifting, phone use go undetected
- No platform connects skill gap → AI-generated learning roadmap → progress tracking → re-matching
- Interview scheduling requires 5+ back-and-forth emails
- Language barriers exclude non-English speakers from quality prep tools

### The Vision
A single platform where a job seeker pastes their LinkedIn URL or uploads a resume, gets semantically matched jobs, sees exactly what skills they lack, practises with a **voice-based AI that speaks questions, detects eye movement, flags tab switches, cross-examines weak answers**, gets a detailed improvement report with a timestamped replay, and schedules real interviews — all in one place, all free.

## 2. USP Features

> These features do **NOT exist** on LinkedIn, Naukri, Indeed, Glassdoor, HireVue, or Pramp.

### 🔗 USP 1: LinkedIn Profile Auto-Import
Paste your LinkedIn public profile URL → backend scrapes full profile data (experience, skills, education, certifications, recommendations count) using Puppeteer/Playwright in headless mode. No manual typing. No resume upload needed. Updates automatically when LinkedIn changes.

### 🎙️ USP 2: AI Voice Mock Interview with Real-Time Cross-Examination
The AI **speaks questions aloud** via Web Speech API TTS. The candidate answers verbally. Groq transcribes + evaluates in real-time. If an answer is weak (<6/10), the AI **automatically generates a follow-up cross-examination question** ("You mentioned Docker — can you explain how you handled a container networking issue?") and speaks it. No other platform does real-time AI cross-examination.

### 👁️ USP 3: Interview Integrity Suite (Camera + Eye Tracking + Tab Detection)
During mock interviews:
- **Camera stays on** via WebRTC — face visible throughout
- **Eye tracking** via face-api.js or MediaPipe FaceMesh — detects when candidate looks away from screen for >3 seconds
- **Tab switch detection** — `visibilitychange` + `blur` events flag cheating
- **Phone detection** (optional) — via webcam ML model
- After interview → **Integrity Score** shown alongside performance score. This is unique to CareerBridge.

### 🗺️ USP 4: AI-Generated Skill Roadmaps with YouTube Course Scraping
Not just "learn Kubernetes." CareerBridge generates a **4-week, day-by-day learning plan** with:
- Real YouTube videos (scraped via YouTube Data API v3 — free tier)
- Official docs links
- Free course links (freeCodeCamp, roadmap.sh, The Odin Project)
- Mini-projects to build
- Weekly quizzes generated by Groq

Candidate marks progress → match score updates dynamically.

### 📊 USP 5: Live "Am I Qualified?" Score While Browsing Jobs
Each job card shows a **live semantic match score** calculated by Groq against the parsed profile. No competitor does this inline without clicking into the job.

### 🧠 USP 6: Interview Replay with Timestamped AI Coaching
Full transcript stored with timestamps. Candidate can replay the interview like video footage review — seeing: question asked → answer given → AI score → where they went wrong → model answer. Like a football coach reviewing match footage.

### 📝 USP 7: Resume A/B Tester
Upload two resume versions → run both against the same job description → get a diff of what each does better and which scores higher. No competitor has this.

### 📧 USP 8: One-Click Cold Outreach Generator
Select a job → get a personalized cold email to the recruiter + LinkedIn DM + follow-up sequence, generated by Groq using the candidate's actual profile data.

### 🌐 USP 9: Multilingual Interview Support
Candidates can conduct interviews in English, Hindi, Spanish, French, German, Mandarin, Arabic. AI evaluates answers in the chosen language. Groq LLaMA 3.3 supports 30+ languages natively.

### 🎯 USP 10: "Jobs You're Almost Qualified For" Feed
A separate feed showing roles where the candidate is 55–74% matched — one or two skills away. Groq explains exactly what to learn to cross the threshold.

### 🤝 USP 11: Recruiter AI Shortlister with Side-by-Side Comparison
Recruiter selects 2–3 candidates → Groq generates a comparison: skills, experience, interview scores, integrity scores, verdict. Like having a junior HR analyst on demand.

### 📈 USP 12: Market Pulse Dashboard
Aggregated real-time data from all JDs on the platform → "Most in-demand skills this week in Mumbai", "Average React dev salary at 3 YOE", "Companies hiring the most right now."

```javascript
const cors       = require('cors');
const http       = require('http');
const { Server } = require('socket.io');
const helmet     = require('helmet');
const morgan     = require('morgan');
const rateLimit  = require('express-rate-limit');

const app    = express();
const server = http.createServer(app);
const io     = new Server(server, {
  cors: { origin: process.env.FRONTEND_URL, credentials: true }

// Security
app.use(helmet());
app.use(morgan('dev'));
app.use(rateLimit({ windowMs: 15 * 60 * 1000, max: 300 }));
app.use(cors({ origin: process.env.FRONTEND_URL, credentials: true }));
app.use(express.json({ limit: '15mb' }));

// ── Routes
app.use('/api/auth',         require('./routes/auth'));
app.use('/api/resume',       require('./routes/resume'));
app.use('/api/linkedin',     require('./routes/linkedin'));
app.use('/api/jobs',         require('./routes/jobs'));
app.use('/api/interview',    require('./routes/interview'));
app.use('/api/schedule',     require('./routes/schedule'));
app.use('/api/tutor',        require('./routes/tutor'));
app.use('/api/roadmap',      require('./routes/roadmap'));
app.use('/api/quiz',         require('./routes/quiz'));
app.use('/api/applications', require('./routes/applications'));
app.use('/api/dashboard',    require('./routes/dashboard'));
app.use('/api/outreach',     require('./routes/outreach'));
app.use('/api/market',       require('./routes/market'));

// ── Socket.io: Real-time interview events
io.on('connection', (socket) => {
  socket.on('join_interview', (sessionId) => {
    socket.join(sessionId);
    console.log(`Socket joined interview: ${sessionId}`);
  });

  // Eye tracking events from browser
  socket.on('eye_drift', (data) => {
    io.to(data.sessionId).emit('integrity_event', {
      type: 'eye_drift', timestamp: Date.now(), duration: data.duration
    });
  });

  // Tab switch
  socket.on('tab_switch', (data) => {
    io.to(data.sessionId).emit('integrity_event', {
      type: 'tab_switch', timestamp: Date.now()
    });
  });

  socket.on('answer_submitted',  (data) => io.to(data.sessionId).emit('processing', true));
  socket.on('evaluation_done',   (data) => io.to(data.sessionId).emit('next_question', data));
});

app.set('io', io);
server.listen(process.env.PORT || 5000, () =>
  console.log(`🚀 CareerBridge API ready on port ${process.env.PORT || 5000}`)
);
```

**`backend/middleware/auth.js`**
```javascript
const jwt      = require('jsonwebtoken');
const supabase = require('../db/supabase');

const authenticate = async (req, res, next) => {
  const token = req.headers.authorization?.split(' ')[1];
  if (!token) return res.status(401).json({ error: 'No token provided' });
  try {
    const { userId } = jwt.verify(token, process.env.JWT_SECRET);
    const { data, error } = await supabase
      .from('profiles').select('*').eq('id', userId).single();
    if (error) throw error;
    req.user = data;
    next();
  } catch {
    res.status(401).json({ error: 'Invalid or expired token' });
  }
};

const requireRole = (...roles) => (req, res, next) =>
  roles.includes(req.user?.role)
    ? next()
    : res.status(403).json({ error: 'Insufficient permissions' });

module.exports = { authenticate, requireRole };
```

**`backend/routes/auth.js`**
```javascript
const router   = require('express').Router();
const jwt      = require('jsonwebtoken');
const supabase = require('../db/supabase');
const { authenticate } = require('../middleware/auth');

router.post('/register', async (req, res) => {
  const { email, password, full_name, role = 'candidate', company_name } = req.body;

  const { data, error } = await supabase.auth.admin.createUser({
    email, password, email_confirm: true,
  });
  if (error) return res.status(400).json({ error: error.message });

  const userId = data.user.id;
  await supabase.from('profiles').insert({ id: userId, email, full_name, role });

  if (role === 'candidate') {
    await supabase.from('candidate_profiles').insert({ user_id: userId });
  }
  if (role === 'recruiter') {
    await supabase.from('recruiter_profiles').insert({ user_id: userId, company_name });
  }

  const token = jwt.sign({ userId }, process.env.JWT_SECRET, { expiresIn: '7d' });
  res.json({ token, user: { id: userId, email, full_name, role } });
});

router.post('/login', async (req, res) => {
  const { email, password } = req.body;
  const { data, error } = await supabase.auth.signInWithPassword({ email, password });
  if (error) return res.status(401).json({ error: 'Invalid credentials' });

  const { data: profile } = await supabase
    .from('profiles').select('*').eq('id', data.user.id).single();
  const token = jwt.sign({ userId: data.user.id }, process.env.JWT_SECRET, { expiresIn: '7d' });
  res.json({ token, user: profile });
});

router.get('/me', authenticate, (req, res) => res.json({ user: req.user }));

module.exports = router;
```

```javascript
const Groq   = require('groq-sdk');
const groq   = new Groq({ apiKey: process.env.GROQ_API_KEY });
const MODEL  = 'llama-3.3-70b-versatile';
const WHISPER = 'whisper-large-v3';

/**
 * Call Groq and parse JSON response.
 * Retries once if JSON is malformed.
 */
async function groqJSON(systemPrompt, userContent, retries = 0) {
  const res = await groq.chat.completions.create({
    model: MODEL,
    temperature: 0.1,
    max_tokens: 4096,
    messages: [
      {
        role: 'system',
        content: systemPrompt + '\n\nReturn ONLY valid JSON. No markdown. No explanation. Start with { or [.'
      },
      { role: 'user', content: userContent }
    ]
  });

  const text = res.choices[0].message.content
    .replace(/^```json\s*/i, '').replace(/```\s*$/i, '').trim();

  try {
    return JSON.parse(text);
  } catch {
    if (retries < 1) {
      return groqJSON(
        systemPrompt,
        userContent + '\n\n[IMPORTANT: Your previous response was invalid JSON. Return ONLY the JSON object, nothing else.]',
        1
      );
    }
    throw new Error('Groq JSON parse failed: ' + text.slice(0, 200));
  }
}

/**
 * Call Groq for chat completions (non-JSON).
 */
async function groqChat(messages, system = '', temp = 0.7, maxTokens = 2048) {
  const allMessages = system
    ? [{ role: 'system', content: system }, ...messages]
    : messages;

  const res = await groq.chat.completions.create({
    model: MODEL, temperature: temp, max_tokens: maxTokens,
    messages: allMessages
  });
  return res.choices[0].message.content.trim();
}

/**
 * Transcribe audio using Groq Whisper.
 * @param {Buffer} audioBuffer
 * @param {string} language - ISO 639-1 code (en, hi, es, etc.)
 */
async function transcribeAudio(audioBuffer, language = 'en') {
  const { toFile } = require('openai');
  const file = await toFile(audioBuffer, 'audio.webm', { type: 'audio/webm' });
  const res = await groq.audio.transcriptions.create({
    file,
    model: WHISPER,
    language,        // Whisper auto-detects if not specified
    response_format: 'json'
  });
  return res.text;
}

module.exports = { groqJSON, groqChat, transcribeAudio };
```

## 7. Phase 3 — AI Voice Interview Room (H 8–14)

> This is the **flagship feature**. Build it carefully. Test it in Chrome (best Speech API support).

### 7.1 — How the Interview Room Works

1. Candidate selects type, difficulty, language → clicks Start
2. Camera activates via `getUserMedia` — face visible throughout
3. **MediaPipe FaceMesh** loads in browser → tracks eye gaze direction
4. `document.addEventListener('visibilitychange')` catches tab switches
5. Groq generates 10 tailored questions referencing candidate's actual projects
6. **`SpeechSynthesisUtterance`** speaks each question aloud in chosen language
7. **`SpeechRecognition`** (Web Speech API) captures candidate's verbal answer live
8. On submission → audio/text sent to backend → Groq Whisper transcribes → Groq evaluates
9. If score < 6/10 → Groq generates cross-examination follow-up → TTS speaks it
10. After all questions → Groq generates full report → stored → redirect to report page
11. Report includes: per-question scores, radar chart, integrity score, hire recommendation, timestamped replay

```javascript
function getLanguageName(code) {
  const map = {
```

```jsx
// ── Text-to-Speech ──────────────────────────────────────────────────────────
function speak(text, lang = 'en-US', onEnd = null) {
  window.speechSynthesis.cancel()
  const utterance = new SpeechSynthesisUtterance(text)

  // Map our language codes to BCP-47
  const langMap = {
    en: 'en-US', hi: 'hi-IN', es: 'es-ES',
    fr: 'fr-FR', de: 'de-DE', ar: 'ar-SA',
    zh: 'zh-CN', pt: 'pt-BR'
  }
  utterance.lang  = langMap[lang] || 'en-US'
  utterance.rate  = 0.88
  utterance.pitch = 1.05

  const voices   = window.speechSynthesis.getVoices()
  const preferred = voices.find(v => v.lang === utterance.lang && v.localService)
    || voices.find(v => v.lang.startsWith(lang))
  if (preferred) utterance.voice = preferred

  utterance.onend = onEnd
  window.speechSynthesis.speak(utterance)
}

// ── Speech Recognition Hook ─────────────────────────────────────────────────
function useSpeechRecognition(language = 'en') {
  const [transcript,  setTranscript]  = useState('')
  const [isListening, setIsListening] = useState(false)
  const recRef = useRef(null)

  const langMap = {
    en: 'en-US', hi: 'hi-IN', es: 'es-ES',
    fr: 'fr-FR', de: 'de-DE', ar: 'ar-SA',
    zh: 'zh-CN', pt: 'pt-BR'
  }

  const startListening = useCallback(() => {
    const SR = window.SpeechRecognition || window.webkitSpeechRecognition
    if (!SR) { toast.error('Speech recognition not supported. Use Chrome.'); return }

    recRef.current = new SR()
    recRef.current.continuous    = true
    recRef.current.interimResults = true
    recRef.current.lang          = langMap[language] || 'en-US'

    recRef.current.onresult = (e) => {
      const text = Array.from(e.results).map(r => r[0].transcript).join(' ')
      setTranscript(text)
    }
    recRef.current.onerror = (e) => {
      if (e.error !== 'aborted') toast.error(`Mic error: ${e.error}`)
    }
    recRef.current.start()
    setIsListening(true)
  }, [language])

  const stopListening = useCallback(() => {
    recRef.current?.stop()
    setIsListening(false)
  }, [])

  return { transcript, setTranscript, isListening, startListening, stopListening }
}

function useEyeTracking(sessionId, onDrift) {
  const videoRef   = useRef(null)

  const streamRef  = useRef(null)
  const driftTimer = useRef(null)

    try {
      const stream = await navigator.mediaDevices.getUserMedia({
        video: { width: 320, height: 240, facingMode: 'user' },
        audio: false
      })
      streamRef.current = stream
      if (videoRef.current) videoRef.current.srcObject = stream
      setCameraOn(true)

      // Simple eye-direction detection using blink/face bounds
      // In production: load MediaPipe FaceMesh for accurate gaze
      startGazeDetection()
    } catch {
      toast.error('Camera access denied. Eye tracking disabled.')
    }
  }

  function startGazeDetection() {
    // Simplified: detect if user looks away using requestAnimationFrame
    // For full implementation, load @mediapipe/face_mesh
    let awayTime = 0
    const CHECK_INTERVAL = 3000 // ms before flagging drift

    const interval = setInterval(() => {
      // Placeholder: in real impl, check FaceMesh gaze vectors
      // If gaze.x < 0.2 or gaze.x > 0.8 → looking away
      const isLookingAway = false // Replace with MediaPipe result
      if (isLookingAway) {
        awayTime += 1000
        if (awayTime >= CHECK_INTERVAL) {
          onDrift?.(awayTime)
          awayTime = 0
        }
      } else {
        awayTime = 0
      }
    }, 1000)

    driftTimer.current = interval
  }

    clearInterval(driftTimer.current)
    setCameraOn(false)
  }

  return { videoRef, canvasRef, cameraOn, startCamera, stopCamera }
}
```

```javascript
const axios = require('axios');

const YT_BASE = 'https://www.googleapis.com/youtube/v3';
const API_KEY = process.env.YOUTUBE_API_KEY;

/**
 * Search YouTube for free learning resources for a skill.
 * Returns top 10 videos sorted by relevance + view count.
 */
async function searchYouTubeCourses(skill, maxResults = 8) {
  const queries = [
    `${skill} tutorial for beginners 2024`,
    `${skill} complete course free`,
    `${skill} crash course`,
  ];

  const allVideos = [];

  for (const q of queries.slice(0, 2)) {
    const { data } = await axios.get(`${YT_BASE}/search`, {
      params: {
        part:       'snippet',
        q,
        type:       'video',
        maxResults: 5,
        videoDuration: 'medium', // 4-20 min — tutorials, not short clips
        videoDefinition: 'high',
        key: API_KEY,
      }
    }).catch(() => ({ data: { items: [] } }));

    allVideos.push(...(data.items || []).map(item => ({
      video_id:     item.id.videoId,
      title:        item.snippet.title,
      channel:      item.snippet.channelTitle,
      description:  item.snippet.description?.slice(0, 200),
      thumbnail:    item.snippet.thumbnails?.medium?.url,
      url:          `https://www.youtube.com/watch?v=${item.id.videoId}`,
      published_at: item.snippet.publishedAt,
    })));
  }

  // Deduplicate by video_id
  const seen = new Set();
  return allVideos.filter(v => {
    if (seen.has(v.video_id)) return false;
    seen.add(v.video_id);
    return true;
  }).slice(0, maxResults);
}

/**
 * Get video statistics (views, likes) for a list of video IDs.
 */
async function getVideoStats(videoIds) {
  if (!videoIds.length) return [];
  const { data } = await axios.get(`${YT_BASE}/videos`, {
    params: { part: 'statistics,contentDetails', id: videoIds.join(','), key: API_KEY }
  }).catch(() => ({ data: { items: [] } }));

  return data.items || [];
}

module.exports = { searchYouTubeCourses, getVideoStats };
```

```javascript
async function evaluateQuizAnswers({ skill, week, questions, candidateAnswers }) {
  const results = questions.map((q, i) => ({
    question_id:      q.id,
    question:         q.question,
    correct_answer:   q.correct_answer,
    candidate_answer: candidateAnswers[i],
    is_correct:       String(candidateAnswers[i]).toLowerCase() ===
                      String(q.correct_answer).toLowerCase(),
    explanation:      q.explanation,

                      String(q.correct_answer).toLowerCase() ? q.points : 0,

  }));

  const total_possible = results.reduce((s, r) => s + r.points_possible, 0);
  const score_percent  = Math.round((total_earned / total_possible) * 100);
```

```javascript
const { groqJSON } = require('../services/groq/client');
const { authenticate } = require('../middleware/auth');

// Aggregate real job data into market insights
router.get('/pulse', authenticate, async (req, res) => {
  const { location } = req.query;

  const { data: jobs } = await supabase.from('job_postings')
    .select('required_skills, salary_range, location, job_category, created_at')
    .eq('is_active', true)
    .order('created_at', { ascending: false })
    .limit(200);

  // Aggregate skill demand
  const skillCount = {};
  jobs?.forEach(job => {
    (job.required_skills || []).forEach(skill => {
      const name = skill.name || skill;
      skillCount[name] = (skillCount[name] || 0) + 1;
    });
  });

  const topSkills = Object.entries(skillCount)
    .sort((a, b) => b[1] - a[1])
    .slice(0, 15)
    .map(([name, count]) => ({ name, count, demand: count > 30 ? 'High' : count > 10 ? 'Medium' : 'Growing' }));

  res.json({
    top_skills:      topSkills,
    total_jobs:      jobs?.length || 0,
    by_category:     groupBy(jobs, 'job_category'),
    by_remote:       groupBy(jobs, 'remote_policy'),
    updated_at:      new Date(),
  });
});

function groupBy(arr, key) {
  return (arr || []).reduce((acc, item) => {
    const val = item[key] || 'Other';
    acc[val] = (acc[val] || 0) + 1;
    return acc;
  }, {});
}

module.exports = router;
```

```javascript
const router   = require('express').Router();
const supabase = require('../db/supabase');
const { authenticate, requireRole } = require('../middleware/auth');
const nodemailer = require('nodemailer');

router.post('/slots', authenticate, requireRole('recruiter'), async (req, res) => {
  const { slots } = req.body;
  const inserts = slots.map(s => ({
    recruiter_id: req.user.id,
    slot_datetime: s.datetime,
    duration_minutes: s.duration_minutes || 45
  }));
  const { data } = await supabase.from('scheduling_slots').insert(inserts).select();
  res.json(data);
});

router.get('/slots/:recruiter_id', authenticate, async (req, res) => {
  const { data } = await supabase.from('scheduling_slots')
    .select('*').eq('recruiter_id', req.params.recruiter_id)
    .eq('is_booked', false).gte('slot_datetime', new Date().toISOString()).order('slot_datetime');
  res.json(data);
});

router.post('/book', authenticate, requireRole('candidate'), async (req, res) => {
  const { slot_id, job_id, notes } = req.body;
  const { data: slot } = await supabase.from('scheduling_slots')
    .select('*').eq('id', slot_id).single();

  if (slot.is_booked) return res.status(409).json({ error: 'Slot already booked' });

  const meetLink = `https://meet.google.com/cb-${Date.now().toString(36)}`;

  await supabase.from('scheduling_slots').update({ is_booked: true }).eq('id', slot_id);
  const { data } = await supabase.from('scheduled_interviews').insert({
    slot_id, candidate_id: req.user.id,
    recruiter_id: slot.recruiter_id, job_id, meet_link: meetLink, notes
  }).select().single();

  // Email notification
  const transporter = nodemailer.createTransport({
    service: 'gmail',
    auth: { user: process.env.GMAIL_USER, pass: process.env.GMAIL_PASS }
  });

  const dateStr = new Date(slot.slot_datetime).toLocaleString('en-IN');
  await transporter.sendMail({
    from: `CareerBridge AI <${process.env.GMAIL_USER}>`,
    to: req.user.email,
    subject: '✅ Interview Scheduled — CareerBridge AI',
    html: `<h2>Your interview is confirmed!</h2>
           <p><strong>Date:</strong> ${dateStr}</p>
           <p><strong>Duration:</strong> ${slot.duration_minutes} minutes</p>
           <p><strong>Meet Link:</strong> <a href="${meetLink}">${meetLink}</a></p>
           <p>Prepare well and use your CareerBridge AI mock interview score!</p>`
  }).catch(e => console.warn('Email error:', e.message));

  res.json({ scheduled: data, meet_link: meetLink });
});

router.get('/my', authenticate, async (req, res) => {
```
